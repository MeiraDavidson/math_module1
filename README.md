## How the Colab links work

Each badge points to
`https://colab.research.google.com/github/MeiraDavidson/math_module1/blob/main/notebooks/<notebook>.ipynb`.
Colab loads the notebook straight from this repository, so any update pushed to
`main` is reflected the next time a reader opens the link.
